Here I have added 5 views of Products. Please finad the details about them.


List View
From this view user can check the available products. User can add new products with details from this view.
Grid View
This shows the detailed information of available products. This is readonly view.
Chart View
This view allows you to compare the available products with their values. This view is only for visualization.
Table View
This view allows you to view the list available products with their details. This view has ~1M records.
Library View
In this view, Template is define in 'ngx-product-list' library and accepts data in specified formate.
